# Sales Data Analysis
This project demonstrated data analysis using Python (Pandas & Matplotlib) on raw sales data.

# Key Insights
- Customers prefer Credit card payments
- Revenue is concentrated in Lisbon under Manager Joao Silva.
- Revenue shows a seasonal spike in November and December.
- Beverages sell the most by volume but Burgers brings in more revenue.
- A small number of managers drive the majority of revenue.

 # How To Run
 1. Download the repository.
 2. Open 'python project-sales.ipynb' in Jupiter Notebook.
 3. Run all cells to reproduce the analysis.
