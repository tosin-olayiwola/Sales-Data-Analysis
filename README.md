# Sales Data Analysis
This project demonstrated data analysis using Python (Pandas & Matplotlib) on raw sales data to uncover business insights and trends that can drive data informed decisions.

---

## Key Insights
- **Credit cards** are the preferred payments method among customers.
- **Lisbon** generates the highest revenue under Manager Joao Silva.
- **November and December** show the strong seasonal spike in revenue.
- **Beverages** sell the most by volume while **Burgers** contribute higher revenue.
- A small number of managers drive the majority of revenue.

---

## Tools Used
- **Python** (Pandas & Matplotlib)
- **Excel** (for initial data inspection)

---

## Data Cleaning & Analysis Steps
1. Loaded and explored the raw dataset.
2. Removed duplicates and handles missing values.
3. Performed grouping and aggregation for summaries.
4. Visualized insights with charts and graphs.

---

## How To Run
 1. Download the repository.
 2. Open 'python project-sales.ipynb' in Jupiter Notebook.
 3. Run all cells to reproduce the analysis.
